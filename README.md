# Web Data Scraping and Analysis Using Excel

## Overview
This project demonstrates web data scraping, cleaning, analysis, and visualization using Microsoft Excel and Power Query.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Charts

## Workflow
1. Scraped product data from a website using Power Query
2. Cleaned and transformed raw data
3. Analyzed price trends using pivot tables
4. Visualized insights using column charts

## Outcome
Built an end-to-end Excel-based data analytics workflow suitable for beginner data analyst roles.
